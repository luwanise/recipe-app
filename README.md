## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/luwanise/recipe-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd recipe-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the app:
   ```bash
   npm start
   ```

## Usage
- Open the app in your browser.
- Explore and search for recipes.
